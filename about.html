ABOUT FOSS

WHAT IS FOSS?

FOSS stands for Free and Open Source Software. It is also known as FLOSS which stands Free/Libre Open Source Software. Libre is a Spanish word meaning 'free'. There is no great difference between the two; FLOSS is simply emphasises the way in which the software is free. This is because many people think 'free' is in terms of price but in reality it is in the sense of freedom of speech, action and removing copyright constraints. FOSS/FLOSS provides a free license to everyone in enabling them to view, copy, edit, modify and reuse the software's source code.

WHAT IS FREE SOFTWARE?

Richard Stallman defines free software as software which provides users liberty to do as they wish with it. Stallman has also provided the world with a criteria of what qualifies as a free software. He believes that any program that meets the following four essential freedoms, is free software:

1. Freedom to operate the program as you desire.

2. Freedom to study the source code of the program and change it in order for the program to do as you wish.

3. Freedom to help your neighbour by redistributing exact copies of the software whenever you would like to.

4. Freedom to contribute to your community by distributing copies or modified versions of the software anytime you want to.

OPEN SOURCE SOFTWARE VS CLOSED SOURCE SOFTWARE

Open source software is code which is accessible to the public. This code is source code rather than executable as it is easier to read. Alternatively, closed source software is code which is not accessible to the public and this is usually unknown because the authority wants us to be controlled as a society. This authoritative act is called centralisation as surveillance on people is habitually controlled by one central authority, especially in the government sphere where it is widely studied and practiced.
Most government bodies and companies, such as Apple, create proprietary closed source software (e.g. iOS) enforcing DRM (Digital Restrictions Management) in which the owner controls the program and the program controls the user. However, FOSS activists are against centralisation and for decentralisation which is the process of distributing/dispersing functions, powers, people/things away from a central location or authority.
